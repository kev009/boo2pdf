Creating an Extended Shelf
 
To create a new extended shelf: 
  1. Under Shelf Title, enter a title of your new shelf - (required)
  2. Under Shelf Name, enter a shelf name. This is the name stored internally in the shelf. (required, limited to 8 characters)
  3. Under Shelf Author, enter a shelf author (this is optional, freeform text.
  4. Under Shelf Index Name, enter the name of any pre-built shelf index that was built for this shelf (e.g. if for a shelf called "mybooks" the index file name is called mybooks.bki, the Index Name should be specified as "mybooks" - (optional, but recommended, limited to 8 characters)
  5. Under Shelf Index Dictionary, select a language code:

  6. Under the Shelf Index Update Timestamp , enter the update date and time when the shelf was last updated. The format is MM/dd/YY HH:mm:ss, where MM is the month (1-12), dd is the day of the month and YY is the two digit year, HH is the hour (0-23), mm is the minute and ss is the second.
  7. From the Include Books From Locations window you will see a list of user specified paths and directories, and the number of books in each directory. Check the boxes next to the path names on your system, or attached to your system via a network, that include books you want to include in the bookshelf you are creating.  All of the books from the directories you selected will be listed in the Include Selected Books window. 

    8.  
  8. From the Include Selected Publications window, select the publications you want to include in your new shelf: 
    • Single publication - click on the book 
    • Multiple publications in various places - hold down the Ctrl key, then click on each publication wanted 
    • Multiple publications in sequence - hold down the Shift key, select the first publication in the list and last in the list 
    • All books - press the Ctrl + A keys 
  9. Click OK to save the shelf. 

Using the Create Bookshelf Window Buttons
 
OK Save the shelf and close the Create Extended Shelf window. A Save Shelf window will appear with the file name you specified for the shelf. Click Save to create and save the shelf.
Cancel Close the Create Extended Shelf window without creating the shelf.
