Navigating between Notes
A list of all notes in the book appears in the upper panel of the Notes window. The topic ID and the first line of each note appears in the list. Double-click on a note to see the full text for the note under Note Body in the lower panel of the Notes window. The Book Reader will now be positioned at the topic in which the note appears. The body of a note may be edited in the Note Body panel, but you must save the changes (to avoid losing them) before another note is displayed.

Using the Toolbar

Create New Note
Opens an input window to enter a new note on the currently-displayed topic.
Delete Note
Deletes the currently-selected note.

Save Note
Saves any changes made to the currently-displayed note.
Previous Note
Moves to the previous note in the list and displays the note text under the Note Body section.  The Book Reader will not be positioned to the note topic
Next Note
Moves to the next note in the list and displays the note text under the Note Body section.  The Book Reader will not be positioned to the note topic.
Help
Displays this help screen.
