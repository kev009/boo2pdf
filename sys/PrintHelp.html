IBM Softcopy Reader

Print

 
Current

This allows you to print the topic currently open in Book Reader.

  1. To start the print process from the Menu Bar: Click the File button, then click Print.
  2. Click Current. The Print window will open, allowing you to specify how and where you want to print the current topic.
Selected Topics
  1. All of the topics you selected for printing will be displayed one after the other in the Book Reader window.
  2. The Print window will open allowing you to specify how and where you want to print the selected topics.
Range of Topics
  1. Select the topics you want to print from the Table of Contents by highlighting (clicking on) them. Use your keyboard keys to help select topics - hold down the Shift key to select a list of topics or hold down the Ctrl key to select specific topics.
  2. To start the print process from the Menu Bar: Click the File button, then click Print.
    • All topics within the range of the outermost selected topics will be displayed one after another in the Book Reader window.
    • The Print window will open allowing you to specify how and where you want to print the specified range of topics.
Cancel

This will cancel the print process.
